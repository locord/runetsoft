Как вариант рассмотрел бы Redis для хранения:
- Есть счетчик из коробки, https://redis.io/commands/incr
- Атомарный, не нужны блокировки
- Быстрый, так как работае в памяти
- Умеет сохранять данные на диск

Для уникальности можно хранить ключ для инкремента в функции и
/**
$banner - объект баннера
$user - объект пользователя, если показываются для авторизованного пользователя
*/
public function showedBannerCacheKey($banner, $user)
{
    return sprintf("users.%s.showed.%s", $user->id, $banner->id);
}
Также можно использовать cookies для идентификации браузера, но нужно будет следить с корректностью принимаемых данных,
т.к куки можно легко подделать
